# Semana 1 - Ejercicios con NumPy

En este trabajo realicé una serie de ejercicios prácticos utilizando la librería NumPy en Python, enfocados en el uso de arrays y matrices con google colab.

## Contenidos trabajados de acuerdo a las pautas de esta semana de trabajo

- Creación de arrays con ceros
- Generación de secuencias numéricas
- Uso de números pares
- Creación de matrices con valores aleatorios
- Operaciones básicas entre arrays (suma)
- Multiplicación de matrices
- Cálculo de valores estadísticos (máximo, mínimo y promedio)
- Obtención de diagonales
- Suma de filas y columnas en matrices

## Archivo que incluí

- `Semana1_numpy.ipynb`

## Herramientas que utilicé

- Python
- NumPy
- Google Colab
